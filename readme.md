
# VRMモデル＆アニメーション テストツール

## 概要
このツールは、VRM形式の3Dアバターモデル（.vrm）とVRMA形式のアニメーションファイル（.vrma）をWebブラウザ上で読み込み、再生・動作確認できるWebアプリケーションです。  

---

## 特徴

- VRMファイル（.vrm）をローカルからアップロードして表示
- VRMAアニメーションファイル（.vrma）を最大15個まで同時に読み込み
- アニメーションの選択、再生、停止
- 口パク、まばたき動作の制御

---

## ツールの起動方法

本ツールの実行にはPython環境が必要です。あらかじめPythonの実行環境を構築してください。

1. コマンドライン（コマンドプロンプト、ターミナル等）を起動します。
2. コマンドラインより、`cd`コマンドを用いて本ツールに移動します。  
    cd "本フォルダのパス"
3. 以下のPythonコマンドでローカルサーバーを起動し、ライブラリを利用可能な状態にします。  
    python -m http.server 8000
4. Webブラウザ上で以下にアクセスし、本ライブラリを表示します。  
    http://localhost:8000

---

## 使い方

1. ファイル選択

「VRMファイル」ボタンで .vrm ファイルを選択
「VRMAファイル」ボタンで .vrma ファイル（最大15個）を選択

2. 読み込み

「モデル&アニメ読み込み」ボタンを押すと、モデルとアニメーションが読み込まれます

3. アニメーション再生

「アニメーション選択」リストから再生したいアニメを選ぶ
「再生」ボタンでアニメーションを再生、「停止」ボタンで停止

4. 口パク・まばたき

「口パク開始」「口パク終了」ボタンで口パクのON/OFF
「まばたき開始」「まばたき終了」ボタンでまばたきのON/OFF

5. 透過画像保存

「背景透過で保存」ボタンで、現在表示されているVRMモデルをポーズ含め背景透過で保存
